# MOKOS

## Apa itu MOKOS ?

MOKOS adalah suatu website yang dirancang untuk mengelola informasi seperti kamar, penyewaan, dan penghuni kos. Selain itu website ini juga bisa digunakan untuk booking kamar kos secara online.

## Dibuat menggunakan :

- HTML
- CSS
- JavaScript
- CodeIgniter 4
- Bootstrap 4
- MySQL
